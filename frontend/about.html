Avenor AI

An enterprise-grade multi-agent generative AI platform. Multiple specialized agents collaborate through an orchestrator to deliver intelligent, grounded, and multimodal responses.

🤖 Agent Architecture
🧠

Orchestrator Agent

Routes and decomposes user queries. Coordinates all agents and assembles the final response pipeline.

🔍

Research Agent

Performs Retrieval-Augmented Generation (RAG) to fetch relevant knowledge from ingested documents.

📊

Analysis Agent

Uses an LLM to reason over the retrieved context and generate structured, high-quality responses.

🎯

Decision Agent

Classifies the query and determines the optimal routing path through the agent ecosystem.

💾

Memory Agent

Maintains short-term in-session memory of all queries for context continuity.

⚡

Action Agent

Handles direct action-based and factual queries using context from the Research Agent.

🎨

Image Agent

Generates images using Stable Diffusion (via Hugging Face API) based on user prompts.

🧪 Core AI Concepts

Agentic AI

Multiple autonomous agents with dedicated responsibilities — reasoning, planning, and collaborating to handle complex multi-step tasks beyond the scope of single-model systems.

Retrieval-Augmented Generation (RAG)

Documents are embedded into a FAISS vector store. At query time, semantically similar chunks are retrieved and passed as context to the LLM, dramatically reducing hallucinations.

Multimodal AI

Avenor processes both text and visual modalities. The Image Agent integrates diffusion-based models for generating contextual visual outputs from natural language prompts.

Workflow Orchestration

The Orchestrator decomposes every query into a structured workflow: Memory → Research → Decision → Analysis/Action/Image → Response assembly.
