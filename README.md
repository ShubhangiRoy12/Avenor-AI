# 🚀 Avenor AI — Enterprise Multi-Agent GenAI Platform

Avenor AI is a next-generation enterprise-grade multi-agent artificial intelligence platform designed to simulate collaborative intelligence using specialized AI agents. Unlike traditional chatbot systems that rely on a single monolithic response pipeline, Avenor AI decomposes complex queries into orchestrated workflows handled by dedicated agents responsible for research, reasoning, memory, decision-making, and multimodal generation.

The platform integrates:

- 🧠 Agentic AI  
- 🤖 Generative AI  
- 📚 Retrieval-Augmented Generation (RAG)  
- 🎨 Diffusion-Based Image Generation  
- 🔍 Semantic Retrieval  
- 🧩 Workflow Orchestration  
- 🌌 Multimodal AI Systems  

to create a scalable enterprise-style intelligent assistant capable of contextual reasoning, intelligent routing, and visual content generation.

---

# 🌌 Core Features

## 🧠 Multi-Agent Architecture
Avenor AI uses multiple autonomous agents working collaboratively instead of a single-response AI pipeline.

## 🔍 Retrieval-Augmented Generation (RAG)
Integrated FAISS vector database with semantic retrieval for grounded and context-aware responses.

## 🤖 LLM-Powered Reasoning
Uses Groq-hosted Llama models for intelligent reasoning, summarization, and contextual response generation.

## 🎨 AI Image Generation
Generates AI visuals and posters using Stable Diffusion through Hugging Face Inference APIs.

## 💾 Conversational Memory
Maintains short-term conversational continuity through a dedicated memory agent.

## ⚖️ Intelligent Routing
Queries are dynamically classified and routed to the most appropriate agent pipeline.

## 🌐 Premium Enterprise UI
Modern futuristic dark-mode dashboard with glassmorphism, smooth animations, workflow visualization, and responsive design.

---

# 🏗️ Agent Architecture

| Agent | Responsibility |
|---|---|
| 🧠 Orchestrator Agent | Coordinates and manages the complete AI workflow |
| 🔍 Research Agent | Retrieves contextual knowledge using RAG |
| 📊 Analysis Agent | Performs LLM-powered reasoning and response synthesis |
| ⚖️ Decision Agent | Dynamically routes queries to correct pipelines |
| 💾 Memory Agent | Maintains conversational memory and context |
| ⚡ Action Agent | Handles lightweight factual and direct-response tasks |
| 🎨 Image Agent | Generates multimodal AI visuals and posters |

# 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/avenor-ai.git
cd avenor-ai
```

---

## 2️⃣ Create Virtual Environment

### 🪟 Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### 🍎 Linux / Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 🔑 Environment Variables

Create a `.env` file in the root directory:

```env
GROQ_API_KEY=your_groq_api_key
HF_API_TOKEN=your_huggingface_api_token
```

---

# ▶️ Run Backend

```bash
uvicorn backend.main:app --reload
```

Backend runs at:

```text
http://127.0.0.1:8000
```

Swagger Docs:

```text
http://127.0.0.1:8000/docs
```

---

# 🌐 Run Frontend

Use VS Code Live Server or any local static server.

Frontend:

```text
http://127.0.0.1:5500/frontend/index.html
```

---

# 📚 RAG Setup

Add `.txt` documents inside:

```text
data/docs/
```

Then build FAISS index:

```bash
python backend/rag/ingest.py
```
